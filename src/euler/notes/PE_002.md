With the Fibonacci sequence starting with $F_1 = 1$ and $F_2 = 2$, it follows that  
$$F_1 \equiv 1 \pmod{2}$$ 
$$F_2 \equiv 0 \pmod{2}$$
$$F_3 = F_1 + F_2 \equiv 1 \pmod{2}$$
$$F_4 = F_2 + F_3 \equiv 1 \pmod{2}$$
$$F_5 = F_3 + F_4 \equiv 0 \pmod{2}$$ 
and so on...  
Therefore $F_i \equiv 0 \pmod{2}$ iff $i \equiv 2 \pmod{3}$.  
This means we can add every third fibonacci number to the sum without having to check if it or any other numbers are even.
