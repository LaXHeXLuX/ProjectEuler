# [PE_0034](../PE_0034.java)

Done by recursively generating all digit factorial sums and tracking the number of digits used, $d$. If $d$ is so large that even adding $(B-1)!$ to the sum wouldn't get it over $B^{d-1}$, prune.

`<B> <time in ms>: <sum of digit factiorials>`
