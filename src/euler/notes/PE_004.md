# [PE_004](../PE_004.java)

Given a number of digits $n$, we want to search for factors $f_1$ and $f_2$, such that the product $f_1f_2$ is palindromic and maximised.  
[Without loss of generality](https://en.wikipedia.org/wiki/Without_loss_of_generality), $f_1 \le f_2$, so $10^{n-1} \le f_1 \le f_2 < 10^n$.  
When a solution $x$ is found, there is no point checking any $f_2 < \sqrt{x}$, as  
$f_1 \le f_2 \Rightarrow f_1f_2 \le f_2^2$, and $f_2 < \sqrt{x} \Rightarrow f_2^2 < x \Rightarrow f_1f_2 < x$.
