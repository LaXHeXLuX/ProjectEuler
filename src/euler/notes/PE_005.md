In order to divide all numbers from $1$ to $20$, a number $x$ really only has to be divisible by $16$, $9$, $5$, $7$, $11$, $13$, $17$, and $19$ (each maximum power of a prime so that it doesn't exceed $20$).  
Every other number $d \le 20$ will also divide $x$, because all $d$-s prime factors already satisfy the requirement.

Then, to find the first number that is divisible by all numbers from $1$ to $n$, we must find the product of $p^{\lfloor \log_pn \rfloor}$ for each prime $p \le n$.
