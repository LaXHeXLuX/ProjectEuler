# [PE_0064](../PE_0064.java)

Given an integer $N$, you need to find the period length of the continued fraction of $\sqrt{N}$. Let $a_0$ be the whole part of $\sqrt{N}$, then $\sqrt{N} = a_0 + \frac{\sqrt{N} - a_0}{1}$.  
Let $a = a_0$, $m = a_0$, and $d = 1$, to get the expression $a + \frac{\sqrt{N} - m}{d}$. Our goal is to find the recursive formula to get $a'$, $m'$, and $d'$, such that $a + \frac{\sqrt{N} - m}{d} = a + \frac{1}{a' + \frac{\sqrt{N} - m'}{d'}}$.  
$$a + \frac{\sqrt{N} - m}{d} = a + \frac{1}{\frac{d}{\sqrt{N} - m}} = a + \frac{1}{\frac{d(\sqrt{N} + m)}{N - m^2}}$$
Since at first $d = 1$, we can assert that $d$ divides $N - m^2$. We will show that this is then also true for $d'$ and $m'$, proving by induction that it is always true. Let $d' = \frac{N - m^2}{d}$. Then 
$$a + \frac{1}{\frac{d(\sqrt{N} + m)}{N - m^2}} = a + \frac{1}{\frac{\sqrt{N} + m}{d'}}$$
Let $a' = \lfloor \frac{a_0 + m}{d'} \rfloor$ and $m' = a'd' - m$. Then
$$a + \frac{1}{\frac{\sqrt{N} + m}{d'}} = a + \frac{1}{a' + \frac{\sqrt{N} - m'}{d'}}$$
giving us 
$$a + \frac{\sqrt{N} - m}{d} = a + \frac{1}{a' + \frac{\sqrt{N} - m'}{d'}}$$
To assert that $d'$ divides $N - {m'}^2$ (assuming $d$ divides $N - m^2$):
$$N - {m'}^2 = N - {a'}^2{d'}^2 + 2 a' d' m - m^2 = N - m^2 - {a'}^2{d'}^2 + 2 a' d' m =$$
$$= dd' - {a'}^2{d'}^2 + 2 a' d' m = d'(d - {a'}^2d' + 2 a' m)$$
