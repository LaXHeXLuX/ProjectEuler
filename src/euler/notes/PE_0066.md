# [PE_0066](../PE_0066.java)

The problem is related to [Pell's equation](https://en.wikipedia.org/wiki/Pell%27s_equation). Fundamental (i.e. minimal) solutions $(x, y)$ to the equation $x^2 - Dy^y = 1$ are found using the continued fraction for $\sqrt{D}$.
