# [PE_0076](../PE_0076.java)

Let $p(n)$ be the number of ways to partition a set of size $n$.
Per the [pentagonal number theorem](https://en.wikipedia.org/wiki/Pentagonal_number_theorem#Relation_with_partitions):
$$p(n) = \sum_k (-1)^{k-1}p(n - g_k)$$
where the sum goes on as long as $n - g_k$ is positive, where $g_k = \frac{k(3k-1)}{2}$, and where $k_i = (-1)^{i-1} \lceil \frac{i}{2} \rceil$.
