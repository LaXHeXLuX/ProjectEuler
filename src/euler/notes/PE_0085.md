In an $n$ by $m$ grid, given an $i$ by $j$ rectangle, there are exactly $(n - i + 1)(m - j + 1)$ ways to fit it in the grid. Then to find the total number of rectangles contained in a given $n$ by $m$ grid, we can use the formula:
$$f(n, m) = \sum_{i=1}^n \sum_{j=1}^m (n - i + 1)(m - j + 1) =$$
$$= \sum_{i=1}^n (n - i + 1) \sum_{j=1}^m (m - j + 1) =$$
$$= \sum_{i=1}^n i \sum_{j=1}^m j =$$
$$= \frac{nm(n+1)(m+1)}{4} =$$
When trying to find $(n, m)$ such that $f(n, m)$ is as close to a given integer $L$ as possible, we want to minimise $\vert L - f(n, m) \vert$, which is the same as minimizing $\vert 4L - nm(n+1)(m+1) \vert$. We start iterating over $n$, and find $4L = nm(n+1)(m+1) \Rightarrow \frac{4L}{n(n+1)} = m(m+1)$, giving us bounds for $m$ as $\lfloor \sqrt{\frac{4L}{n(n+1)}} \rfloor$ and $\lfloor \sqrt{\frac{4L}{n(n+1)}} \rfloor + 1$.

To lessen search space, we can require $m > n$, meaning
$$m >  \sqrt{\frac{4L}{n(n+1)}} - 1 > n \Rightarrow$$
$$\Rightarrow \frac{4L}{n(n+1)} > n^2 + 2n + 1 \Rightarrow$$
$$\Rightarrow 4L > (n^2 + 2n + 1)(n^2 + n) > n^4 \Rightarrow$$
$$\Rightarrow n < \root{4}\of{4L}$$
