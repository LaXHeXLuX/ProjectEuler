# [PE_0086](../PE_0086.java)

Given a cuboid with lengths $(a, b, c)$, the three shortest path candidates $l_1$, $l_2$, $l_3$ are:
1. $$\sqrt{(a+b)^2 + c^2}$$
2. $$\sqrt{(a+c)^2 + b^2}$$
3. $$\sqrt{(b+c)^2 + a^2}$$

Without loss of generality, we can assume $a \le b \le c$, which means
$$2ab \le 2ac \le 2bc \Rightarrow$$
$$(a+b)^2 + c^2 \le (a+c)^2 + b^2 \le (b+c)^2 + a^2 \Rightarrow$$
$$l_1 \le l_2 \le l_3$$
Then, the shortest path is always $\sqrt{(a+b)^2 + c^2}$. For the shortest path to have integer length, $(a, b, c)$ must satisfy
$$(a+b)^2 + c^2 = x^2$$
for some integer $x$, i.e. $(a+b, c, x)$ must be a Pythagorean triplet. Then $a+b$ and $c$ can't both be odd, since then $x^2 \equiv 2 \pmod{4}$, which isn't possible.

We iterate over $c \le M$, and care about the unique solution count for $a+b$, given $c$. For that, we iterate over $a+b$, and if $(a+b)^2 + c^2$ gives a whole square, we figure out how many valid $(a, b)$ pairs would give $a+b$. The lower bound for $b$ is $\lceil \frac{a+b}{2} \rceil = \lfloor \frac{a+b+1}{2} \rfloor$, since $b \ge a$. Therefore, $b \le $. The upper bound for $b$ is either $a+b-1$ or $c$, whichever is smaller. Then the solution count given $c$ and $a+b$ is $\text{min}(a+b-1, c) - \lfloor \frac{a+b+1}{2} \rfloor + 1 = \text{min}(a+b-1, c) - \lfloor \frac{a+b-1}{2} \rfloor$.
