# [PE_0088](../PE_0088.java)

The upper limit for a minimal product-sum number for a given $k$ is $2k$, since $1^{k-2} \times 2 \times k = (k-2)1 + 2 + k$.  
Given that we only care about $k \le L$ for some limit $L$, we don't care about product-sums that exceed $2L$, since those can never be minimal.

Everything else is simple DP.
