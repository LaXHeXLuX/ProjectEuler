## Pythagorean triplets
As the problem states, a pythagorean triplet is a triplet $(a, b, c)$ which satisfies $a^2 + b^2 = c^2$ (we won't enforce $a < b$).  
From [Pythagorean triple: Generating a triple](https://en.wikipedia.org/wiki/Pythagorean_triple#Generating_a_triple), every $(a, b, c)$ triplet is uniquely defined by integers $k$, $m$, $n$, such that $m > n$, $m+n \equiv 1 \mod{2}$, and $\text{gcd}(m, n) = 1$.  
Let:  
$a = k(m^2 - n^2)$  
$b = k(2mn)$  
$c = k(m^2 + n^2)$  
Then:  
$a^2 + b^2 =$  
$= (k(m^2 - n^2))^2 + (k(2mn))^2 =$  
$= k^2(m^4 - 2m^2n^2 + n^4) + k^2(4m^2n^2) =$  
$= k^2(m^4 + 2m^2n^2 + n^4) =$  
$= (k(m^2 + n^2))^2 =$  
$= c^2$

## The current problem
$P = a+b+c = k(m^2 - n^2) + k(2mn) + k(m^2 + n^2) = 2km(m+n)$.  
This means the sum of a Pythagorean triplet can only be even.  
We can start iterating through potential solutions by finding all divisors of $\frac{P}{2}$ and try $k$ for each value.
Then $\frac{P}{2k} = m(m + n)$. Since $0 < n < m$, $m^2 < m(m + n) < 2m^2$, meaning $\sqrt{\frac{P}{4}} < m < \sqrt{\frac{P}{2}}$.  
Additionally, since $m+n \equiv 1 \mod{2}$, $m \equiv m(m + n) \mod{2} \equiv \frac{P}{2k} \mod{2}$.  
Iterating $m$ with these constraints, $n = \frac{P}{2k} \div m - m$. If an $(m, n)$ pair is found such that $\frac{P}{2k} \equiv 0 \mod{m}$ and $\text{gcd}(m, n) = 1$, it will generate a valid triple.
