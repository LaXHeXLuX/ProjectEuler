# [PE_0091](../PE_0091.java)

## Main algorithm
We want to find triangles $\triangle OPQ$ with $O(0, 0)$, $P(x_1, y_1)$, and $Q(x_2, y_2)$, with $0 \le x_1, y_1, x_2, y_2 \le L$ for some limit $L$.

Let's first consider the trivial cases:
1) if $x_1 = 0$ and $y_2 = 0$, the resulting triangle is always a right triangle. There are $L^2$ such combinations.
2) if $x_1 = 0$ and $y_1 = y_2$, the resulting triangle is always a right triangle. There are $L^2$ such combinations. The same goes for when $y_2 = 0$ and $x_1 = x_2$, giving $2L^2$ such combinations in total.

This gives $3L^2$ trivial cases in total.
There is also a semi-trivial case: when $x_1 = 0$ and $Q(x_2, y_2)$ lies on the semicircle between $O$ and $P$, i.e. $x_2^2 + (y_2-\frac{y_1}{2})^2 = \frac{y_1^2}{4} \Rightarrow x_2^2 = y_2(y_1 - y_2)$. We iterate over $y_2$ and check if $x_2$ is a perfect square. For each hit, there is also an additional solution with the hypotenuse on the $x$-axis.

This only leaves cases where $1 \le x_1, y_1, x_2, y_2 \le L$.

Without loss of generality, say we want $\angle OPQ = \frac{\pi}{2}$. Then $\theta = \frac{\pi}{2} \Rightarrow \text{cos}(\theta) = 0$, meaning $PO \cdot PQ = 0$. $PO = (-x_1, -y_1)$, and $PQ = (x_2-x_1, y_2-y_1)$, giving $y_2 = \frac{x_1(x_1 - x_2)}{y_1} + y_1$

## Additional speedups

Since we've taken care of the trivial cases, there can be no solutions where $x_1 = x_2$, because then $y_2 = \frac{0(0 - 0)}{y_1} + y_1 = y_1$, and $P(x_1, y_1) = Q(x_2, y_2)$ don't give triangles. Then we can require $x_1 < x_2$, and simply add $2$ for each found solution.

We can also force $y_2 = \frac{x_1(x_1 - x_2)}{y_1} + y_1$ to be in the acceptable $1 \le y_2 \le L$ range. Solving the quadratic equation, we get bounds $\frac{L - \sqrt{L^2 - 4x_1(x_1 - x_2)}}{2} \le y_1 \le \frac{L + \sqrt{L^2 - 4x_1(x_1 - x_2)}}{2}$.
