# [PE_0100](../PE_0100.java)

With $n$ total discs and $b$ blue discs, the probability of getting two blue discs is $\frac{b}{n} \cdot \frac{b-1}{n-1} = \frac{b(b-1)}{n(n-1)}$. For the probability to be exactly $50\%$, we need $2b(b-1) = n(n-1) \Rightarrow n^2 - n - (2b^2 - 2b) = 0$. Using the quadratic formula, we get
$$n = \frac{1 \pm \sqrt{1 + 8b^2 - 8b}}{2}$$
Since $n$ needs to be positive, we can simplify the $\pm$ to a $+$:
$$n = \frac{1 + \sqrt{8b^2 - 8b + 1}}{2}$$
For $n$ to have an integer solution, we need $8b^2 - 8b - (m^2 - 1) = 0$ for some integer $m$. Using the quadratic formula, we get
$$b = \frac{8 \pm \sqrt{64 + 32m^2 - 32}}{16}$$
Since $b$ needs to be positive, we can simplify the $\pm$ to a $+$:
$$b = \frac{2 + \sqrt{2m^2 + 2}}{4}$$
For $b$ to have an integer solution, we need $x^2 - 2m^2 = 2$ for some integer $x$ - a generalized Pell equation. Iterating over solutions, we want $n > L$ for some limit $L$. $n = \frac{m + 1}{2}$, meaning:
1) $m$ must be odd
2) $m > 2L - 1$

However, since $x^2 - (2m_0)^2 = 2$ would mean $x^2 \equiv 2 \pmod 4$, which isn't possible, then $m$'s oddness is guaranteed, and we do not need to check for it.

Once $m > 2L - 1$, the answer is $b = \frac{x + 2}{4}$. Again, $x \equiv 0 \pmod{2}$ is guaranteed.
