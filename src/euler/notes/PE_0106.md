For a set of size $n$, the count of possible subset pairs is
$$\frac{\sum_{i=1}^{n-1} \binom{n}{i} \sum_{j=1}^{n-i} \binom{n-i}{j}}{2}$$

The division by $2$ is necessary since the ordering in a subset pair doesn't matter. 

If we can assume the second rule is already satisfied, then we don't need to check any subset pairs where one has more elements than the other, since the one with more elements is guaranteed to have a larger sum. That reduces the possible subset pair count to
$$\sum_{i = 1}^{\frac{n}{2}} \frac{\binom{n}{i} \binom{n-i}{i}}{2} = \sum_{i = 1}^{\frac{n}{2}} \binom{n}{2i} \frac{\binom{2i}{i}}{2}$$

For $n = 4$, that is, given a strictly increasing set $(a_1, a_2, a_3, a_4)$, we can describe any subset pair with the indexes of the elements inside those subsets, i.e. $(a_1, a_3), (a_2, a_4) \Rightarrow (1, 3), (2, 4)$. Note that for this example, we see that $1 < 2$ and $3 < 4$, meaning the first subset will certainly have a smaller sum than the second, therefore we don't need to test for it. In fact, we don't need to test for any subset pairs where each index of one subset can be paired with a larger index of another. 

So the question becomes:  
Given $2n$ elements, mark $n$ elements as `(`, and $n$ elements as `)`, based on whether they will be in the first or second subset. Given such an arrangement, is it a valid ordering of parenthesis? If it is, that means that for each `(`, there is a "bigger" `)`. But if not, then there is at least one `(` which can't be paired up with a "bigger" `)`.
