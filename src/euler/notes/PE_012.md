# PE_012

A triangle number $T(n)$ is of the form $\frac{n(n+1)}{2}$.  
Instead of computing the divisors of the full number, we can use the fact that $n$ and $n+1$ are co-prime.
1) If $n \equiv 0 \mod{2}$, $\text{divisorCount}(T(n)) = \text{divisorCount}(\frac{n}{2}) \cdot \text{divisorCount}(n+1)$
2) If $n \equiv 1 \mod{2}$, $\text{divisorCount}(T(n)) = \text{divisorCount}(n) \cdot \text{divisorCount}(\frac{n+1}{2})$
