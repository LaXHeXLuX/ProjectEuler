# [PE_014](../PE_014.java)

Let $C(n)$ mean the Collatz chain length of $n$.  
Given a limit $L$, we can only check numbers $\frac{L}{2} \le n < L$, since for any $n < \frac{L}{2}$, there is $2n < L$, which will necessarily have $C(2n) = C(n) + 1$.  
Also, any $n \equiv 4 \pmod{6}$ can be skipped, since for any $n = 6k + 4$, there exists $2k + 1$ which will necessarily have $C(2k+1) = C(6k + 4) + 1$.
