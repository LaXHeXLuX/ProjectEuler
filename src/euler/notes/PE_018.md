# PE_018

The maximal path sum can be found layer by layer, bottom-up. Starting from the second to last row, it is easier to see that in order to maximise the path sum, out of the two numbers available in the last row, the path should finish on the bigger number. Then we might as well eliminate the last row and add to each second-to-last row's element the bigger number it could access. Doing this repeatedly will eventually leave us with only the top-most element, which now has the value of the maximal path sum.
