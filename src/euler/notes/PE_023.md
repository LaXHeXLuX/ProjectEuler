The solution takes advantage of the fact that every even number higher than $46$ can be written as two abundant numbers.  

## Proof

Lemma 1: all numbers $6k$, $k > 1$, are abundant.  
Proof: for any $k$, $6k$ has divisors $k$, $2k$, $3k$. For $k > 1$, $1$ is also a separate divisor. So the sum of divisors $S(6k) \ge 1 + k + 2k + 3k = 6k + 1 > 6k$ ◻

Then, for each even number $2n$ where $2n \ge 24$ and $n \equiv 0 \pmod{3}$, $2n = 6k_1 + 6k_2$ for $k_1 = 2$ and $k_2 = \frac{n}{3} - 2$.  
That leaves cases $n \equiv 1 \pmod{3}$ and $n \equiv 2 \pmod{3}$.

Lemma 2: all numbers $20k$, $k > 0$, are abundant.  
Proof: for any $k$, $20k$ has divisors $k$, $2k$, $4k$, $5k$, $10k$. So the sum of divisors $S(20k) \ge k + 2k + 4k + 5k + 10k = 22k > 20k$ ◻

We want to satisfy $2n = 6k_1 + 20k_2$ for $k_1 > 1$ and $k_2 > 0$, or $n = 3k_1 + 3 + 10k_2$ for $k_1 > 0$ and $k_2 > 0$.  
Then $k_1 = \frac{n - 3 - 10k_2}{3}$, which means $n \pmod{3} \equiv 10k_2 \pmod{3} \equiv k_2 \pmod{3}$.

1) $n \equiv 1 \pmod{3}$. Let $k_2 = 1$. Then $k_1 = \frac{n - 13}{3}$, which is a positive integer for $n > 13$.
2) $n \equiv 2 \pmod{3}$. Let $k_2 = 2$. Then $k_1 = \frac{n - 23}{3}$, which is a positive integer for $n > 23$.

In all cases, the conditions are satisfied for $2n > 46$ ∎

## Algorithm

First, using a sieve of divisor sums, a boolean lookup array of all abundant numbers is created. Odd abundant numbers are also stored in a list.  
For $n \le 46$, $n \equiv 0 \pmod{2}$, we check by brute force whether there exists an abundant number pair summing to $n$.  
For $n \equiv 1 \pmod{2}$, we loop over odd abundant numbers and check if the difference is abundant.
