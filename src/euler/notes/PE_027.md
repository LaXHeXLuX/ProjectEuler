# [PE_027](../PE_027.java)

For $f(n) = n^2 + an + b$, $f(0) = b$ and $f(1) = 1 + a + b$ must both be prime to get a nontrivial score of consecutive primes.  
$f(b) = b^2 + bn + b = b \cdot (b + n + 1)$ is not prime. Then if $|a| < L$ and $|b| \le L$ for some limit $L$, the maximum formula value that can potentially be prime is $f(L-1) = (L-1)^2 + (L-1)^2 + L = 2L^2 - 3L + 1$ with $a = L-1$ and $b = L$.  
We generate a lookup array for all primes up to $2L^2 - 3L + 1$, and additionally generate an array of primes up to $2L$ (the maximum value for $f(1)$).

We iterate over the array of primes to get $b$ (breaking after reaching $L$), satisfying $f(0)$ primality. Secondly, we iterate over the array of primes to get $f(1)$, and let $a = f(1) - b - 1$, satisfying $f(1)$ primality. To get the score, we iterate $n$ until $f(n)$ is no longer prime.
