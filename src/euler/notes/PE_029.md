# [PE_029](../PE_029.java)

Given $a_1$ and $a_2$, write $a_1 = a_{1_0}^{k_1}$ and $a_2 = a_{2_0}^{k_2}$, where $k_1, k_2 \ge 1$, and $a_{1_0}, a_{2_0}$ are not perfect powers. Then 
$$a_1^{b_1} = a_2^{b_2} \Leftrightarrow (a_{1_0}^{k_1})^{b_1} = (a_{2_0}^{k_2})^{b_2} \Leftrightarrow a_{1_0}^{k_1b_1} = a_{2_0}^{k_2b_2} \Leftrightarrow a_{1_0} = a_{2_0} \land k_1b_1 = k_2b_2$$

## Counting distinct $kb$ values

The count of distinct terms $a^b$ for $2 \le a \le L_a$, $2 \le b \le L_b$, is equal to $(L_a - 1)(L_b - 1)$ minus the $(k, b)$ values where there already exists $(k', b')$, $k' < k$, such that $kb = k'b'$. For a given $k$, the possible $kb$ values are $V(k) = [2k, 3k, \ldots, L_bk]$. Given $k' < k$, the possible $k'b'$ values are $V(k') = [2k', 3k', \ldots, L_bk']$, we need to find the count of duplicates in $V(k)$ and $V(k')$, i.e. $V(k) \cap V(k')$.

Duplicate elements are necessarily of the form $\text{lcm}(k, k') \cdot n$ for some integer $n$. Since values bigger than $L_bk'$ won't appear in $V(k')$, $n \le \frac{L_bk'}{\text{lcm}(k, k')}$. Since values smaller than $2k$ won't appear in $V(k)$, $n \ge \frac{2k}{\text{lcm}(k, k')}$, and $n > \frac{2k - 1}{\text{lcm}(k, k')}$, giving us the total count of possible $n$ values as
$$\lfloor\frac{L_bk'}{\text{lcm}(k, k')}\rfloor - \lfloor\frac{2k - 1}{\text{lcm}(k, k')}\rfloor$$
Then, given $k$ and $b$, the number of unique $kb$ values not seen for any $k' < k$, is found using the [Inclusion–exclusion principle](https://en.wikipedia.org/wiki/Inclusion%E2%80%93exclusion_principle) against all $V(k')$.

## Algorithm

Start with $\text{count} = 0$, $\text{highSkip} = 0$. Iterating over $2 \le a_0 \le \sqrt{L_a}$, we:
1) Check if $a_0$ is a perfect power. If yes, continue to next $a_0$
2) For each power $p$ of $a_0$, $a_0^p = a$, while $a \le L_a$:
   1) if $a \le \sqrt{L_a}$, mark $a$ as not a perfect power
   2) if $a > \sqrt{L_a}$, increment $\text{highSkip}$
   3) add distinct $kb$ to $\text{count}$

Each $a$, $\sqrt{L_a} < a \le L_a$, where $a$ is not a perfect power, has exactly $L_b - 1$ unique $a^b$ values. We add $(L_a - \sqrt{L_a} - \text{highSkip}) \cdot (L_b - 1)$, where the first $\sqrt{L_a}$ are already checked, and there are $\text{highSkip}$ perfect powers between $\sqrt{L_a}$ and $L_a$.
