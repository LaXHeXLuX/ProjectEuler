# [PE_032](../PE_032.java)

Let $d_B(n)$ mean "number of digits in $n$ in base $B$". We will prove that in any base $b$, given $a \cdot b = c$, $a < b$, one
of two cases is true:

1) $d_B(c) = d_B(a) + d_B(b) - 1$
2) $d_B(c) = d_B(a) + d_B(b)$

## Proof

Lemma 1: $d_B(n) = \lfloor\log_B(n)\rfloor + 1$.  
Proof: $d_B(n)$ is equivalent to asking "What is the integer $d$ such that $b^{d-1} \le n < b^d$?".
Taking $\log_B$ of all sides
gives $d-1 \le \log_B(n) < d \Rightarrow d-1 = \lfloor \log_B(n) \rfloor \Rightarrow d = \lfloor \log_B(n) \rfloor + 1$ ◻

Lemma 2: Given $a, b > 0$, one of two cases is true:

1) $\lfloor a + b \rfloor = \lfloor a \rfloor + \lfloor b \rfloor$
2) $ \lfloor a + b \rfloor = \lfloor a \rfloor + \lfloor b \rfloor + 1$

Proof: Let $a = a_0 + a'$ and $b = b_0 + b'$, such
that $a_0 = \lfloor a \rfloor$, $b = \lfloor b \rfloor$, $0 \le a', b' < 1$.
Then $\lfloor a \rfloor + \lfloor b \rfloor = a_0 + b_0$ and $\lfloor a + b \rfloor = \lfloor a_0 + a' + b_0 + b' \rfloor = a_0 + b_0 + \lfloor a' + b' \rfloor$.  
We know that $0 \le a' + b' < 2$. That leaves two cases:

1) $0 \le a' + b' < 1 \Rightarrow \lfloor a' + b' \rfloor = 0 \Rightarrow \lfloor a + b \rfloor = a_0 + b_0 + 0 = \lfloor a \rfloor + \lfloor b \rfloor$
2) $1 \le a' + b' < 2 \Rightarrow \lfloor a' + b' \rfloor = 1 \Rightarrow \lfloor a + b \rfloor = a_0 + b_0 + 1 = \lfloor a \rfloor + \lfloor b \rfloor + 1$

◻

From the given $a \cdot b = c$, taking $\log_B$ of both sides
gives $\log_B(a \cdot b) = \log_B(a) + \log_B(b) = \log_B(c)$.  
By lemma 2, $\lfloor \log_B(c) \rfloor$ is equal to
either $\lfloor \log_B(a) \rfloor + \lfloor \log_B(b) \rfloor$ or $\lfloor \log_B(a) \rfloor + \lfloor \log_B(b) \rfloor + 1$.  
Then  $\lfloor \log_B(c) \rfloor + 1$ is equal to
either $\lfloor \log_B(a) \rfloor + 1 + \lfloor \log_B(b) \rfloor + 1 - 1$ or $\lfloor \log_B(a) \rfloor + 1 + \lfloor \log_B(b) \rfloor + 1$.  
By lemma 1, $d_B(c)$ is equal to either $d_B(a) + d_B(b) - 1$ or $d_B(a) + d_B(b)$ ∎

## Algorithm

In order for numbers $a, b, c$, $a \cdot b = c$ to be a pandigital group in a given base $B$, it follows that
$$d_B(a) + d_B(b) + d_B(c) = B - 1$$
Then one of two cases is true:
1) $d_B(c) = d_B(a) + d_B(b) - 1 \Rightarrow$
   $d_B(c) + 1 = d_B(a) + d_B(b) \Rightarrow$  
   $d_B(c) + 1 + d_B(c) = B - 1 \Rightarrow$  
   $d_B(c) = \frac{B}{2} - 1$ and $d_B(a) + d_B(b) = \frac{B}{2}$
2) $d_B(c) = d_B(a) + d_B(b)$
   $d_B(c) + d_B(c) = B - 1 \Rightarrow$  
   $d_B(c) = \frac{B - 1}{2}$ and $d_B(a) + d_B(b) = \frac{B - 1}{2} + 1$

Case 2 is only valid (and unique from case 1) when $B \equiv 1 \pmod{2}$.

We start by iterating $a$. Since $a < b$, $d_B(a) \le d_B(b)$. Going over the two cases:
1) $d_B(a) + d_B(b) = \frac{B}{2} \Rightarrow$  
   $2 \cdot d_B(a) \le \frac{B}{2} \Rightarrow$  
   $d_B(a) \le \lfloor \frac{B}{4} \rfloor$
2) $d_B(a) + d_B(b) = \frac{B - 1}{2} + 1 \Rightarrow$  
   $2 \cdot d_B(a) \le \frac{B - 1}{2} + 1 \Rightarrow$  
   $d_B(a) \le \lfloor \frac{\lfloor \frac{B}{2} \rfloor + 1}{2} \rfloor$

In either case, continue by iterating over $b$, with the constraint $B^{d_B(c)-1} \le a \cdot b < B^{d_B(c)}$ and check if $a \cdot b$ is new and $(a, b, a \cdot b)$ is a pandigital group. For the last one, a bitmask is useful.
