# [PE_0321](../PE_0321.java)

The minimum number of moves $M(n)$ is achieved by first moving $1$ red counter, then $2$ blue counters, then $3$ reds, etc. until all counters of a given color are moving. Then $n$ red counters are moved, then $n-1$ blue counters, etc. until the positions are reversed. This takes
$$n + 2\sum_{i=1}^n i = n(n+2)$$
moves.

If $M(n)$ is a triangle number, then
$$2n(n+2) = x(x+1)$$
for some integer $x$. Rearranging gives
$$2n^2 + 4n - (x^2 + x) = 0$$
for which the quadratic formula gives
$$n = \frac{-2 \pm \sqrt{2x^2 + 2x + 4}}{2}$$
Then
$$2x^2 + 2x - (m^2-4) = 0$$
for some integer $m$, for which the quadratic formula gives
$$x = \frac{-1 \pm \sqrt{2m^2 - 7}}{2}$$
Then $y^2 - 2m^2 = -7$ for some integer $y$ - a Pell equation. Iterating through solutions for $(y, m)$ also gives solutions for $(n, x)$.
