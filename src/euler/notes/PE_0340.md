# [PE_0340](../PE_0340.java)
 
Given the crazy function:
- $F(n) = n-c, n > b$
- $F(n) = F(a + F(a + F(a + F(a + n)))), n <= b$

We assume $a > c$, as it greatly simplifies the problem.  
Let's note some examples about the crazy function:
- $F(a + n) = a + n - c, a+n > b$
- $F(a + n) = F(a + F(a + F(a + F(2a + n)))), a+n \le b$
- $F(2a + n) = F(a + F(a + F(a + F(3a + n)))), 2a+n \le b$

It becomes apparent that if $n < b$ and $b-n = qa + m$ with $m < a$ (so $q = \lfloor \frac{b-n}{a} \rfloor$), then 
$$F(n) = F_a^{\circ 3(q+1)}((q+1)a + n)$$
where $F_a^{\circ k}(x)$ denotes recursion $k$ times with the addition of $a$, e.g.
$$F_a^{\circ 3}(x) = F(a + F(a + F(a + x)))$$
$F((q+1)a + n) = (q+1)a + n - c$, and for $3(q+1)$ times, $a-c$ is added, giving
$F(n) = n + (4q+4)a - (3q+4)c$ if $n \le b$

With $S(a, b, c) = \sum_{n=0}^b F(n)$, $\sum_{n=0}^b F(n)$ can be simplified as:
$$\sum_{n=0}^b F(n) =$$
$$= \sum_{n=0}^b (n + (4q+4)a - (3q+4)c) =$$
$$= \sum_{n=0}^b (n + 4qa - 3qc) + \sum_{n=0}^b 4(a-c) =$$
$$= \sum_{n=0}^b n + \sum_{n=0}^b ((4a - 3c) \cdot \lfloor \frac{b-n}{a} \rfloor) + 4(b+1)(a-c) =$$
$$= \frac{b(b+1)}{2} + 4(b+1)(a-c) + (4a - 3c) \sum_{m=0}^b \lfloor \frac{m}{a} \rfloor$$
To find a closed-from expression for $\sum_{m=0}^b \lfloor \frac{m}{a} \rfloor$, note that in the sum, the first $a$ terms will be zero, the next $a$ terms will be one, and so on, up to $m = aq$ in $b = aq + r$, $r < a$. This first part can be expressed as $0 \cdot a + 1 \cdot a + \ldots + (q-1) \cdot a = a \frac{q(q-1)}{2}$. Then what's left is $r+1$ terms where $\lfloor \frac{m}{a} \rfloor = \lfloor \frac{b}{a} \rfloor$, meaning the total sum $S(a, b, c)$ is:
$$\frac{b(b+1)}{2} + 4(b+1)(a-c) + (4a - 3c) (a \frac{q(q-1)}{2} + (r+1)q)$$
where $q = \lfloor \frac{b}{a} \rfloor$ and $r = b - aq$.
