# [PE_0346](../PE_0346.java)
 
Any number $n$ is a repunit $R(2)$ in base $n-1$. Only the number $1$ can ever be repunit $R(1)$. Then for any $n > 1$, it is a repunit if there is a base $b$, where $n$ is $R_b(i)$, $i \ge 3$. 

The first brute-force way to find all integers $n$ under a limit $L$ that are repunits $R_b(i)$, $i \ge 3$ in some base $b$. For $L = 10^{12}$, this is slow.

The second way is to iterate over all bases $b < \sqrt{L}$ and check for all repunits $R_b(i)$, $i \ge 3$ that do not exceed $L$, and save them to a set to avoid duplicate values. $b < L$ is justified because a repunit of length $3$ or more is bigger than $b^2$. This is faster, but not ideal.

The third option is first add $R_b(3)$ for all valid bases $b$, and then iterate over $b < \root{3}\of{L}$, then iterate over $R_b(i)$, $i \ge 4$, checking if it can be an $R(3)$ in some base, and if not, check if we already have it in the set. This introduces two sub-problems: finding a closed-from expression of the sum of all valid $R(3)$ repunits under the limit, and algebraically checking if an $R(i)$ is also an $R(3)$ in some base.
1) For $R_b(3) < L$  in some base $b$, we need $b^2 + b + 1 \le L \Rightarrow b \le \frac{\sqrt{4L - 3} - 1}{2}$ (let the limit for $b$ be $l$). The sum of all $R(3)$ is 
$$\sum_{b = 2}^l b^2 + b + 1$$
    Which, by separating $b^2$, $b$, and $1$, can be rewritten as 
$$\frac{l(l+1)(2l+1)}{6} + \frac{l(l+1)}{2} + l - 3$$
2) Given an $R(i)$, we need to check if there exists an integer solution for $b^2 + b + 1 = R(i)$, i.e. if
$$b = \frac{\sqrt{4R(i) - 3} - 1}{2}$$
is an integer.
