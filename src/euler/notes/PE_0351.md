# [PE_0351](../PE_0351.java)
 
On a hexagonal orchard of order $n$, the six-way symmetry lets us divide the shape into six smaller $n$-size triangles. We can only calculate the hidden points of one such triangle, and multiply by $6$. 

For a given triangle, let a point's location be determined by how many rows away it is from the center ($x$), and which point it is along the triangle row ($y$). Representing it as $\frac{x}{y}$, it becomes apparent that a point $p'$ is hidden by another point $p$ iff $p'$'s "fraction" is reducible to $p$'s "fraction". In fact, any point $p'$ will have some other point between it and the center, iff $p'$'s "fraction" $\frac{x}{y}$ is reducible, i.e. if $\text{gcd}(x, y) > 1$.

Then the point is visible iff $x$ and $y$ are coprime, $\text{gcd} = 1$, and the hidden points can be calculated as
$$6(T(n) - V(n))$$
where $T(n) = \frac{n(n+1)}{2}$ is the number of total points in a triangle, and $V(n)$ is the number of visible points.  
The Euler's totient function $\phi(n)$ will return the count of numbers up to $n$ that are coprime to $n$. Then $V(n)$ is calculated as
$$\sum_{y = 1}^n \phi(y)$$
which is also the [totient summatory function](https://en.wikipedia.org/wiki/Totient_summatory_function), $\Phi(n)$
