# [PE_037](../PE_037.java)

In my wording, a number like $59$ is called left truncatable ($59 \rightarrow 5$) and a number like $283$ is called right truncatable ($283 \rightarrow 83 \rightarrow 3$)

The first digit of any truncatable prime must be prime, i.e. one of $[2, 3, 5, 7]$. The last digit must be prime and not ruin the primality of a larger number, i.e. one of $[3, 7]$. The last digit of any multi-digit prime number must be one of $[1, 3, 7, 9]$.

We recursively generate right truncatable primes, starting with $3$ and $7$, and adding one of $[1, 3, 7, 9]$. At each step, a number is checked if adding any of $[2, 3, 5, 7]$ would give a left truncating prime.

Resulting eleven primes:
```
[23, 53, 313, 73, 373, 317, 37, 3137, 739397, 797, 3797]
```
