# [PE_039](../PE_039.java)

$$a = k(m^2 - n^2)$$
$$b = k(2mn)$$
$$c = k(m^2 + n^2)$$
for some positive integers $k$, $m$, $n$, such that $m > n$, $m+n \equiv 1 \pmod{2}$, and $\text{gcd}(m, n) = 1$.  
Then $p = a + b + c = 2km(m+n)$. If $p \le L$ for a given limit $L$, then $L \ge 2km(m+n) > 2m^2 \Rightarrow m < \sqrt{\frac{L}{2}}$
