# [PE_0401](../PE_0401.java)
 
With $\text{SIGMA}_2(n)$, every divisor $d$ from $1$ to $n$ occurs $\lfloor \frac{n}{d} \rfloor$ times. Therefore,
$$\text{SIGMA}_2(n) = \sum_{d = 1}^n \lfloor \frac{n}{d} \rfloor d^2$$
For each $d$, $\lfloor \frac{n}{d} \rfloor$ will be the same up to $\lfloor \frac{n}{\lfloor \frac{n}{d} \rfloor} \rfloor$. Once $d > \sqrt{n}$, that's more than one value each iteration. Then, for each iteration, the sum to add is 
$$\lfloor \frac{n}{d} \rfloor \sum_{i=d}^{\lfloor \frac{n}{\lfloor \frac{n}{d} \rfloor} \rfloor} i^2$$
We can use the square sum formula for that: $1^2 + 2^2 + \ldots + n^2 = \frac{n(n+1)(2n+1)}{6}$.
