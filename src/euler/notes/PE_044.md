## Algorithm

Let a solution be uniquely described by the integers $i, j, k, l$, where $D = P_i$, $i < k$, $j < k$, $k < l$, and $P_k - P_j = P_i$ and $P_k + P_j = P_l$.
Writing out $P_n = \frac{n(3n-1)}{2}$, we can see that
$$\frac{k(3k-1)}{2} - \frac{j(3j-1)}{2} = \frac{i(3i-1)}{2}$$
which simplifies to
$$3k^2 - k - (3j^2 - j + 3i^2 - i) = 0$$
Using the quadratic formula for $k$, we get
$$k = \frac{1 + \sqrt{1 + 36j^2 - 12j + 36i^2 - i}}{6}$$
For $k$ to be an integer, we need $k = \frac{1 + c}{6}$, where $c^2 = 1 + 36j^2 - 12j + 36i^2 - i$. Using another quadratic formula for $j$, we get
$$j = \frac{1 + \sqrt{a^2 - 36i^2 + 12i}}{6}$$
For $j$ to be an integer, we need $j = \frac{1 + b}{6}$, where $b^2 = c^2 - 36i^2 + 12i$. Using another quadratic formula for $i$, we get
$$i = \frac{1 + \sqrt{1 + c^2 - b^2}}{6}$$
For $i$ to be an integer, we need $i = \frac{1 + a}{6}$, where $a^2 = 1 + c^2 - b^2$. Rearranging gives 
$$(a-1)(a+1) = (c-b)(c+b)$$
This means we can iterate over $i$, get $a = 6i - 1$, and factor the product $p = (a-1)(a+1)$ to iterate over divisors $d_1$ and $d_2 = \frac{p}{d_1}$. Then $c = \frac{d_1 + d_2}{2}$, provided $d_1 \equiv d_2 \pmod{2}$, and $b = \frac{d_2 - d_1}{2}$. If $b \equiv 5 \pmod{6}$ and $c \equiv 5 \pmod{6}$, we only need to check if $P_j + P_k = P_l$ is a valid pentagonal number. If so, we've found a valid solution.

## Speedups
The algorithm has three main bottlenecks: 
1) factoring $p$.
2) spending many iterations where one of the modular checks is bound to fail.
3) spending many iterations where the final $P_l$ check is bound to fail.

I couldn't figure out a way to solve 3, but below are some tricks to minimize the cost of 1 and 2.

### 1. Factoring $p$
Since we know $p$ is of the form $(a-1)(a+1)$, we can factor $a-1$ and $a+1$ separately, them sum the prime factors, and generate the divisors from the factors.

### 2. Pruning invalid iterations
We want to choose $d_1, d_2$, where $d_1 = c-b$ and $d_2 = c+b$, so that $d_1 \equiv d_2 \pmod{2}$, $c \equiv 5 \pmod{6}$, and $b \equiv 5 \pmod{6}$.

By $c$'s and $b$'s modular restrictions, $d_1 \equiv 5-5 \equiv 0 \pmod{6}$ and $d_2 \equiv 5+5 \equiv 4 \pmod{6}$.

First, notice that both $d_1$ and $d_2$ are required to be even. Then we can simply divide $p$ by $4$, factor it, and when iterating the divisors, multiply each by $2$. That also satisfies $d_1 \equiv d_2 \pmod{2}$.

Second, $d_2 \equiv 1 \pmod{3}$, meaning any prime factors $3$ in $p$ must be present in $d_1$, not $d_2$. Similarly as before, we can remove the maximum power of $3$ from $p$, and later multiply $d_1$ by the same power of $3$.
