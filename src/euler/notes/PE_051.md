# [PE_051](../PE_051.java)

To find the smallest possible solution first, we iterate the target number of digits, $d$, in the solution, starting with $d = 2$.  
Secondly, we iterate over the amount of replaceable digits. When the target family size $s > 7$, then we know that the amount of replaceable digits must be divisible by $3$.  
Proof:  
Say that the static part of the number is $x \pmod{3}$. Then, when replacing $r$ replaceable digits with a digit $D$, the whole number is $x+rD \pmod{3}$. For the whole number to be a prime, we want $x + rD \not\equiv 0 \pmod{3} \Rightarrow rD \not\equiv -x \pmod{3}$.  
For $s > 7$ values of $D$, there will be cases where $D \equiv 0 \pmod{3}$, $D \equiv 1 \pmod{3}$, and $D \equiv 2 \pmod{3}$. Then, if $r \not\equiv 0 \pmod{3}$,  there will be also be cases whew $rD \equiv 0 \pmod{3}$, $rD \equiv 1 \pmod{3}$, and $rD \equiv 2 \pmod{3}$, meaning there will always be some $D$ we need to use for which $rD \equiv -x \pmod{3}$. Then $r \equiv 0 \pmod{3}$.
