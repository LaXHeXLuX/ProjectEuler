# [PE_063](../PE_063.java)

For a given base $B$, the power $a^b$ can only have $b$ digits if $a < B$. Also, if $\text{digits}(a^b) < b$ and $a < B$, then also $\text{digits}(a^{b+1}) \le \text{digits}(a^b) + 1 < b+1$.  
This means we only have to check $1 < a < B$, and can stop iterating $1 < b$ once $\text{digits}(a^b) < b$.  
For $1 \le a < B$ and $b = 1$, we will always have $\text{digits}(a^b) = 1 = b$, meaning we can add $B-1$ to the count without iterating.
