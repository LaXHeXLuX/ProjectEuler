# PE_064

The periods length is computed with:
```
private static int length(int n) {
    int a0 = (int) Math.sqrt(n);
    if (a0*a0 == n) return 0;
    int d1 = n - a0*a0;int a1 = 2*a0/d1;
    int m = a1*d1 - a0;
    int d = (n - m*m)/d1;
    int a = (a0 + m)/d;
    int len = 1;
    while (!(m == a0 && d == d1)) {
        len++;
        m = d*a - m;
        d = (n - m*m)/d;
        a = (a0 + m)/d;
    }

    return len;
}
   ```

## Properties
For any positive integer i:
 - $length(x) = 0 \Leftrightarrow x = i*i$
 - $length(x) = 1 \Leftrightarrow x = i*i + 1$
