# PE_070

How to assert that the solution x must be of the form $p_1*p_2$?
1. if $x = p$, then $\phi(x) = p-1$. This minimises $\frac{x}{\phi(x)}$, but numbers differing by 1 can never be permutations.
2. if $x = p^2$, then $\phi(x) = (p-1)*p$. A good proof of why those numbers can never be permutations of one another, can be found [here](https://projecteuler.net/post_id=263) (Rudy, 2004)
3. if $x$ has more than 2 (not necessarily distinct) prime factors:  
    $x = p_1*p_2*...*p_k$,  
    then $\phi(x) \leq (p_1-1)*p_2*...*p_k$.  
    For a given limit $L$, $\phi(x)$ is maximized when $k = 3$ and $p_1 = p_2 = p_3 = L^\frac{1}{k}$, which yields  
    $\phi(x) = (L^\frac{1}{3}-1)*(L^\frac{1}{3})^{2}$.  
    Therefore, having gotten the best fraction with two prime factors, the theoretical limit of $\frac{x}{\phi(x)}$ for more than two factors can be compared. If it is higher than the current solution, then no better solution exists.
