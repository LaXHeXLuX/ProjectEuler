# [PE_072](../PE_072.java)

Given a denominator $d$, the number of reduced proper fractions $\frac{n}{d}$ is equal to the number of integers $n$ where $1 \le n < d$, i.e. equal to $\phi(d)$.

By sieving through all totient values, we can add them for each denominator value.
