# [PE_075](../PE_075.java)

Using the pythagorean triplet theory from [PE_009](PE_009.md).
Given the parametrisation
$$a = k(m^2 - n^2)$$
$$b = k(2mn)$$
$$c = k(m^2 + n^2)$$
for $m \not\equiv n \pmod{2}$ and $\text{gcd}(m, n) = 1$, the perimeter $P$ is
$$P = k(m^2 - n^2) + k(2mn) + k(m^2 + n^2) = 2km(m+n)$$
$m, n, k$ are iterated to form all right triangles under the perimeter limit $L$, and the single-solution perimeters are counted.
