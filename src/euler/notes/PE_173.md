# [PE_173](../PE_173.java)

The tile count in a square laminae can be represented as $(k + 2n)^2 - k^2 = 4kn + 4n^2$, where $k$ is the size of the inner square and $n$ is the width of the laminae.  
Given a limit $L$ and noting $k, n \ge 1$, 
$$4 + 4n^2 \le 4n  + 4n^2 \le 4kn + 4n^2 \le L \Rightarrow n \le \frac{\sqrt{L-4}}{2}$$
Iterating over $n$, the valid bounds for $k$ are $k \ge 1$, and
$$4kn + 4n^2 \le L \Rightarrow k \le \frac{L - 4n^2}{4n}$$
Then the total square laminae count is
$$\sum_{n=1}^{L_n} \lfloor\frac{L - 4n^2}{4n}\rfloor$$
where $L_n = \frac{\sqrt{L-4}}{2}$.  
Let $L = 4n \cdot q + r$, $r < 4n$. $\lfloor\frac{L - 4n^2}{4n}\rfloor = \lfloor\frac{4n \cdot q + r - 4n^2}{4n}\rfloor = \lfloor\frac{4n(q - n) + r}{4n}\rfloor = q-n$, where $q = \lfloor \frac{L}{4n} \rfloor$. Then
$$\sum_{n=1}^{L_n} \lfloor\frac{L - 4n^2}{4n}\rfloor = \sum_{n=1}^{L_n} (\lfloor \frac{L}{4n} \rfloor - n) = \sum_{n=1}^{L_n} \lfloor \frac{L}{4n} \rfloor - \frac{L_n(L_n+1)}{2}$$
However, it seems that $\sum_{n=1}^{L_n} \lfloor \frac{L}{4n} \rfloor$ is impossible to reduce to a closed-form equation for arbitrary $L$.
