# [PE_187](../PE_187.java)

A semiprime under a given limit $L$ is a product of two primes $p_1, p_2$, where at least one is not greater than $\sqrt{L}$.  
We generate all primes up to $\sqrt{L}$ using a sieve. Then, for each prime $p_1$, we need the count of primes $p_2$ such that $p_1 \cdot p_2 < L \Rightarrow p_2 < \frac{L}{p_1}$. We use the same prime counting function as in [PE_010](PE_010.md), with changed $S(L, a)$ to get the count instead of the sum:
$$S(L, 0) = L$$
$$S(0, a) = 0$$
$$S(L, a) = S(L, a-1) - S(\frac{L}{p_{a-1}}, a-1)$$
