# PE_204

Given a type $T$ and a limit $L$, all primes $p$ up to $T$ are generated.  
Inspired by [Assato's comment](https://projecteuler.net/thread=204;page=2#23320), given the general recursive algorithm $\text{hamming}(i, L)$ (where $i$ is the current prime index and $L$ is the upper limit) is:  
1) $\text{hamming}(0, L) = x$, where $2^x \le L < 2^{x+1}$
2) if $L < p_i$, then $\text{hamming}(i, L) = \text{hamming}(i-1, L)$
3) else, $\text{hamming}(i, L) = \text{hamming}(i-1, L) + hamming(i, \frac{L}{p_i}) + 1$

The recursive method uses a memoize table for smaller $L$ values.
