# [PE_205](../PE_205.java)

The chance to roll a sum $x$ with $s$ $n$-sided dice is equivalent to asking:  
What is the number of solutions to $x_1 + x_2 + \ldots + x_s = x$ with $1 \le x_i \le n$ for each $i$.  
We find this by first solving for $x_i \ge 1$, and then subtract solutions where any $x_i > n$.  
The solution count for $x_1 + x_2 + \ldots + x_s = x$, $1 \le x_i$ is equivalent to $y_1 + y_2 + \ldots + y_s = x - s$, $y_i \ge 0$, which is $\binom{x - s + s - 1}{s - 1} = \binom{x - 1}{s - 1}$ (see [Stars and bars](https://en.wikipedia.org/wiki/Stars_and_bars_(combinatorics))).  
Let $S(k)$ note the count of solutions where for at least $k$ $i$, $y_i \ge n$. The count of solutions where any $y_i \ge n$ is found via the [Inclusion–exclusion principle](https://en.wikipedia.org/wiki/Inclusion%E2%80%93exclusion_principle), with:  
$S(1) - S(2) + S(3) - \ldots + (-1)^{s+1}S(s)$  
$S(k) = \binom{s}{k} \cdot \binom{x - kn - 1}{s-1}$ &mdash; there are $\binom{s}{k}$ ways of choosing $k$ dice, and $\binom{x - kn - 1}{s-1}$ ways to divide the remaining "points" after allotting $n$ "points" to $k$ dice.
