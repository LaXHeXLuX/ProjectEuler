# PE_206

The algorithm starts building the $n$ from the right:
1) Start with $\text{place} = 0$ and valid $n$ candidates $[0]$.
2) Let $\text{next}$ be the position of the next defined element of the form from the right. Make a new candidates array as:
   $$\bigcup_{n \,\in\, \text{candidates}} \{n + k \cdot 10^{\text{place}} \mid k \in \{0, 1, \dots, 10^{\text{next}-\text{place}}-1\}\}$$
   For each new candidate $n$, check if $\frac{n^2}{10^\text{next}} \equiv d \pmod{10}$, where $d$ is the $\text{next}$-th element of the form from the right (i.e. `d = form[-next]`). If $\frac{n^2}{10^\text{next}} \not\equiv d \pmod{10}$, toss that $n$. Set $\text{place} = \text{next}$ and repeat until $\text{place} = |\text{form}|$ (i.e. `place = len(form)`) or no candidates remain.
