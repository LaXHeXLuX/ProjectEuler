With Fibonacci numbers up to $F_n$, the biggest sum of nonconsecutive terms is made with $F_x + F_{x-2} + F_{x-4} + \ldots + F_{b}$ where 
$$b = \begin{cases} 2 & \text{if}\ x \equiv 0 \pmod{2} \\ 1 & \text{otherwise} \end{cases}$$  
This sum is always less than  
$$F_{x+1} =$$
$$= F_{x} + F_{x-1} =$$
$$= F_{x} + F_{x-2} + F_{x-3} =$$
$$F_{x} + F_{x-2} + F_{x-4} + \ldots + F_{b} + 1$$
Therefore the Zeckendorf representation of $n$ must include the largest $F_{x}$ that is not larger than $n$. Take for example $100$. Taking away $F_{10} = 89$ gives $11$. Now we need to get the Zeckendorf representation of $11$. Taking away $F_{5} = 8$ gives $3$, which is itself a Fibonacci number $F_{3}$. If we start with a Fibonacci number, e.g. $F_{7} = 21$, $21$ itself is the one-element Zeckendorf representation.

Using this, we can make some deductions about the sum of all $z(n)$ in the range $[n_0, n_L)$, represented as $S(n_0, n_L)$. First, 
$$S(0, n) = S(0, F_x) + S(F_x, n)$$
where $F_x$ is the largest Fibonacci number lower than $n$. Then, 
$$S(F_x, n) = S(0, n - F_x) + n - F_x$$
because for each $i$ in $[F_x, n)$, $z(i) = z(i - F_x) + 1$. Lastly, 
$$S(0, F_x) = S(0, F_{x-1}) + S(0, F_{x-2}) + F_{x-2}$$
which naturally follows from the previous two, taking into account that $F_x - F_{x-1} = F_{x-2}$.
