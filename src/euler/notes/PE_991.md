Let $(a, b, c) = (ЁЯНО, ЁЯНМ, ЁЯНН)$ and $a+b+c \le L$ for some limit $L$. The equation
$$\frac{a}{b+c} + \frac{b}{a+c} + \frac{c}{a+c} = 4$$
can be simplified to
$$3c^2 + c(2b + 3a) - (a^2 + b^2 - 4ab) = 0$$
Using the quadratic formula, we get
$$c = \frac{-2b - 3a \pm \sqrt{s}}{6}$$
Where 
$$s = (2b + 3a)^2 + 12(a^2 + b^2 - 4ab) = 16b^2 - b(36a) + 21a^2$$
Since $\frac{-2b - 3a - \sqrt{s}}{6} \le 0$ and $c > 0$, we can restrict $\pm$ to $+$.  

For $c$ to have a positive integer solution, we need $\sqrt{s} > 2b + 3a$, $-2b - 3a + \sqrt{s} \equiv 0 \pmod{6}$, and $s = x^2$ for some integer $x \ge 0$.

1) $\sqrt{s} > 2b + 3a \Rightarrow s > 4b^2 + 12ab + 9a^2 \Rightarrow 12b^2 - b(48a) + 12a^2 > 0$.  
    If we try to solve for $12b^2 - b(48a) + 12a^2 = 0$, we get 
    $$b = \frac{48a \pm \sqrt{(48a)^2 - 4 \cdot 12 \cdot 12a^2}}{24} = a(2 \pm \sqrt{3})$$
    Since for $b = 2a$, $12b^2 - b(48a) + 12a^2 \le 0$, we need either $0 < b < a(2 - \sqrt{3})$ or $a(2 + \sqrt{3}) < b$, ensuring $b \le L-a$ in both cases.
2) $-2b - 3a + \sqrt{s} \equiv 0 \pmod{6} \Rightarrow \sqrt{s} \equiv 2b + 3a \pmod{6}$. Squaring both sides gives 
   $$s = 16b^2 - b(36a) + 21a^2 \equiv 4b^2 + 3a^3 \equiv (2b + 3a)^2 \pmod{6}$$
   meaning $-2b - 3a + \sqrt{s} \equiv 0 \pmod{6}$ is guaranteed (<span style="color: red">???</span>). 
3) $s = x^2 \Rightarrow 16b^2 - b(36a) + (21a^2 - x^2) = 0$, which gives
   $$b = \frac{36a \pm \sqrt{(36a)^2 - 4 \cdot 16(21a^2 - x^2)}}{32} = \frac{9a \pm \sqrt{4x^2 - 3a^2}}{8}$$
   For $b$ to have an integer solution, we need $9a \pm \sqrt{4x^2 - 3a^2} \equiv 0 \pmod{8}$ and $4x^2 - 3a^2 = y^2$ for some integer $y \ge 0$.
   1) $9a \pm \sqrt{4x^2 - 3a^2} \equiv 0 \pmod{8} \Rightarrow$  
      $9a \equiv \mp \sqrt{4x^2 - 3a^2} \pmod{8} \Rightarrow$  
      $81a^2 \equiv 4x^2 - 3a^2 \pmod{8} \Rightarrow$  
      $4a^2 \equiv 4x^2 \pmod{8} \Rightarrow$  
      $a^2 \equiv x^2 \pmod{2} \Rightarrow$  
      $a^2 \equiv 16b^2 - b(36a) + 21a^2 \pmod{2} \Rightarrow$  
      $4a^2 \equiv 4ab \pmod{2}$, which is guaranteed (<span style="color: red">???</span>).
   2) Rearranging gives us
      $$(2x-y)(2x+y) = 3a^2$$
      Meaning we must iterate over divisors $d$ of $3a^2$ such that $(d, \frac{3a^2}{d})$ can be represented as $(2x-y, 2x+y)$.
