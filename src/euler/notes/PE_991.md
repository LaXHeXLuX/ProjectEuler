# [PE_206](../PE_206.java)

Let $(a, b, c) = (🍎, 🍌, 🍍)$. The equation
$$\frac{a}{b+c} + \frac{b}{a+c} + \frac{c}{a+c} = 4$$
can be simplified to
$$3c^2 + c(2b + 3a) - (a^2 + b^2 - 4ab) = 0$$
Using the quadratic formula, we get
$$c = \frac{-2b - 3a \pm \sqrt{s}}{6}$$
Where 
$$s = (2b + 3a)^2 + 12(a^2 + b^2 - 4ab) = 16b^2 - b(36a) + 21a^2$$
Since $-2b - 3a - \sqrt{s} < 0$ and $c > 0$, we can restrict $\pm$ to $+$.  

For $c$ to have an integer solution, we need $s \ge 0$ and $s = x^2$ for some integer $x$.

1) $s \ge 0 \Rightarrow 16b^2 - b(36a) + 21a^2 \ge 0$. If we try to solve for $16b^2 - b(36a) + 21a^2 = 0$, we get 
    $$b = \frac{36a \pm \sqrt{(36a)^2 - 4 \cdot 16 \cdot 21a^2}}{32} = \frac{9a \pm a\sqrt{-3}}{8}$$
    which doesn't have a real solution. Then, since $s$ is a continuous function, it is always either negative or positive.  
    For $a = 1, b = 1$, $s(a, b) = 16b^2 - b(36a) + 21a^2 = 1$, so $s \ge 0$ is guaranteed.
2) $s = x^2 \Rightarrow 16b^2 - b(36a) + (21a^2 - x^2) = 0$, which gives
    $$b = \frac{36a \pm \sqrt{(36a)^2 - 4 \cdot 16(21a^2 - x^2)}}{32} = \frac{9a \pm \sqrt{64m^2 - 3a^2}}{8}$$
